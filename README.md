 - `Fedora 31 x86_64`:

 - `VM x86` (from __`Windows`__ via `VMWare Player 15` with `Ubuntu 16.04.6 i686`):
   - `gcc 5.4.0 20160609  (Ubuntu 5.4.0-6)`
   - `glibc 2.26.1`
